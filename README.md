## Technologies
Project is created with:
* React
* Redux/Redux Toolkit
* TypeScript
* styled-components
* react-router-dom
* react-toastify
* Material UI
* ESLint
* Prettier

## How to install

1. Clone the repo:

   ```sh
   git clone https://github.com/dmtrhrytsak/blog-app.git
   ```
2. Install packages:

   ```sh
   npm install
   ```
3. Run the app

   ```
   npm start
   ```
